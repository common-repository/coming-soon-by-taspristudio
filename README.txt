== Installation ==

There is no prerequisites for installing **Coming Soon by TaspriStudio** plugin. You can follow the steps below to complete the installation.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen from your WordPress website dashboard.
2.  Search for `Coming Soon by TaspriStudio`.
3.  Install and activate the plugin, that's it!

= Manual Installation =
1.  Download [Coming Soon by TaspriStudio](https://downloads.wordpress.org/plugin/ts-wordpress-coming-soon.zip "Coming Soon by TaspriStudio") plugin.
2.  Extract the `ts-wordpress-coming-soon.zip` file. You'll get plugin files inside `ts-wordpress-coming-soon` directory.
3.  Upload the plugin files to the `/wp-content/plugins/ts-wordpress-coming-soon` directory.
4.  Activate the plugin through the 'Plugins' screen form your WordPress website dashboard.
